Google Maps Lead Scraper

A Python script that collects business lead data from Google Maps and exports it to a clean Excel file.

 What it does
- Prompts the user to enter a city and business category
- Fetches up to 20 local business results
- Exports Name, Rating, Address, Phone, and Website to a formatted .xlsx file

 Built with
- Python
- SerpApi (Google Maps API)
- requests
- openpyxl

 Usage
1. Clone the repo
2. Install dependencies: pip install requests openpyxl
3. Add your SerpApi key to the script
4. Run: python maps_scraper.py
